# 📱 Ordinis

## EN

> Daily reflections to order the soul and draw closer to Jesus Christ.

### ✨ About

**Ordinis** is a mobile application focused on Christian reflections inspired by Saint Augustine and a life in communion with Christ.

Its purpose is simple:  
to provide a daily moment of pause, reflection, and spiritual alignment.

In a fast-paced and distracted world, Ordinis invites you back to what truly matters:
- silence
- truth
- ordered love
- communion with God

---

### ⚙️ Tech Stack

- **Framework:** Flutter / React Native
- **Language:** Dart / TypeScript
- **State Management:** Riverpod / Zustand
- **Storage:** Local JSON + SQLite / AsyncStorage
- **UI:** Custom theming + responsive layout
- **Architecture:** Modular & scalable (feature-based)

---

### 📦 Libraries & Tools

- Navigation (Flutter Navigator / React Navigation)
- Local storage (Hive / AsyncStorage)
- Fonts & UI (Google Fonts / custom typography)
- Notifications (Firebase Cloud Messaging — planned)

---

### 🚀 Features

- Daily reflection system
- Modern interpretation layer
- Favorites & bookmarking
- Content sharing
- Offline-first experience
- Dark mode support
